# Introduction

Nexus Backend is a powerful, extensible backend service for building conversational AI applications.

## Overview

Nexus Backend provides:

- **REST API** — Manage projects, personas, stages, classifiers, knowledge bases, and more
- **WebSocket API** — Real-time conversational AI sessions with streaming audio/text support
- **Authentication** — JWT-based authentication with role-based permissions and API keys
